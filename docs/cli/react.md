# `no-mistakes react`

Analyze React component traits.

| Leaf command | Purpose |
| --- | --- |
| [`react analyze`](react-analyze.md) | Print component traits for matching files. |
| [`react check`](react-check.md) | Fail on requested React trait violations. |

Shared options: `--root`, `--config`, `--format json|yml|md|paths|human`, and
`--json`.
